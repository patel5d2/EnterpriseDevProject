ABOUT THE PROJECT

Built for students,
by students.

The Tri-State Student Directory was created to solve the problem of fragmented student information across universities in the Ohio, Kentucky, and Indiana region. Our goal: one searchable, verified network for every student to connect, collaborate, and grow.

WHY WE BUILT IT

The Problem We're Solving

Students across the tri-state region ‚Äî at institutions like the University of Cincinnati, Xavier University, Northern Kentucky University, and Cincinnati State ‚Äî had no centralized way to discover and connect with peers at other universities. Administrators similarly lacked a reliable cross-institutional directory for communication and outreach.

This application was developed as part of an Enterprise Application Development course to demonstrate a production-ready, full-stack Java web application with real-world features: secure authentication, database-driven search, input validation, and clean REST APIs.
